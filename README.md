# Customer-Spending-Engagement-Analysis-using-Statistical-Modeling
Through this Project i analyzed a dataset of a customers details , which gives insights about customers and their Engagement with the marketplace plateform.

Performed end-to-end statistical analysis on 10,675+ customer transactions to evaluate spending behavior, engagement patterns, and demographic impact. Applied EDA, visualization, correlation analysis, hypothesis testing (t-test, ANOVA, chi-square, correlation) to validate business assumptions and deliver data-driven segmentation strategies for marketing and retention optimization.

• Performed descriptive, bivariate and statistical analysis on 10K+ customer records
• Built demographic segmentation using statistical testing
• Applied t-test, ANOVA, chi-square and correlation for hypothesis validation
• Derived marketing and retention insights using statistical inference
